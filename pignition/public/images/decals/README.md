# Bullet-hole decal assets

Add your **transparent PNG** bullet-hole graphics here. The site will use them as decorative decals.

**Expected files:**
- `bullet-hole-1.png` — first variant (e.g. larger impact)
- `bullet-hole-2.png` — second variant
- `bullet-hole-3.png` — third variant

**Guidelines:**
- Transparent background
- Distressed, decal-style look (not realistic damage)
- Dark circular impact with rough outer ring
- Preferred: dirty, rugged, western / poster style

If a file is missing, the site falls back to a simple circular decal SVG.
