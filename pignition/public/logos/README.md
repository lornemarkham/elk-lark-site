# Pignition — Logo concept directions

Three logo concepts for **Pignition** (Southern Outlaw BBQ). Mood: moody, outlaw, cowboy, rustic but premium, dark and masculine.

**Colors:** bone `#f5f0e8` · burnt orange `#c2410c` · black/charcoal `#0f0f0f`

---

## 1. Clean outlaw wordmark
- **File:** `concept-1-wordmark.svg`
- Main: **PIGNITION** (bold, tall, condensed, slight western)
- Subtitle: **SOUTHERN OUTLAW BBQ** (burnt orange)
- Use: primary lockup for site, signage, print.

## 2. Brand mark / monogram
- **File:** `concept-2-monogram.svg`
- Bold **P** in a stamp-style frame; small burnt orange accent.
- Use: favicon, social avatars, stamps, packaging.

## 3. Patch / stamp (secondary)
- **File:** `concept-3-patch.svg`
- Badge shape with tagline and wordmark. Slightly decorative, still clean.
- Use: merch, stickers, hats, socials.

## 4. Whiskey label / outlaw badge
- **Files:** `concept-4-whiskey-label.svg` (vertical), `concept-4b-whiskey-label-horizontal.svg` (horizontal)
- Bold, ornamental, vintage American. Jack Daniel’s / MC patch / smokehouse inspiration. Double-line frame, corner brackets, stars and dot separators.
- Use: whiskey bottle (vertical), smoker lid, leather patch (horizontal).
